About

Welcome to DevBlog — a personal space for sharing thoughts on software development, tech news, and code.

What you'll find here

Posts are organized into three categories: Articles for longer reads, News for quick updates and links, and Code for snippets and technical walkthroughs.

Built with

Pure HTML, CSS, and vanilla JavaScript on the frontend. Posts are stored in Supabase (PostgreSQL) and served directly to the browser — no server required.
